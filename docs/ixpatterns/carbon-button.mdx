IBM Carbon’s Web Components provide the custom element
`<cds-button>` for initiating actions (for example, **Save** or **Delete**).
Choose the visual style with the `kind` attribute—typical values include
`primary`, `secondary`, and `ghost` (see Carbon’s docs and Storybook for details).
